# ROS2 SLAM Autonomous Navigation Robotics Project

The goal of this project is to develop a wheeled ground robot with a digital twin for autonomous navigation and mapping using SLAM. The robot's name is GIGI and will be referred to as such.

## Project Goals
- Simulate robot in Gazebo with SLAM-based mapping and navigation
- Develop a physical robot using Raspberry Pi + LiDAR sensors (possibly using a robot kit for base parts, then building out from there)
- Run the ROS2 simulation and mapping software on the physical robot
- Build toward a full digital twin for autonomous navigation

## Current Status
- ROS2 workspace set up with basic Python and C++ nodes
- Learning ROS2 fundamentals, OOP, and hardware integration

## Stack
- ROS2
- Gazebo
- SLAM Toolbox
- Nav2
- Raspberry Pi
- Python / C++
